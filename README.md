# Overview
Web-based eLeave management system with role-based access (Employee & Supervisor), supporting leave requests, approvals, and real-time leave balance tracking.

# Features
1. Authentication & Role Management (Employee & Supervisor)
2. Leave Request Submission
3. Leave Balance Tracking
4. Leave Approval (Supervisor)
5. Leave History (Employee & Supervisor)
6. Dashboard (Statistics & Summary)


# Tech Stack
- Backend: ASP.NET Core (Razor Pages / Web API)
- Database: MySQL
- Language: C#
- Tools: Visual Studio, Git, GitHub
